# MCLEGACY_UWP
This is MC LEGACY UWP, inspired by Minecraft UWP. This project is is available for Windows 10 1809 or higher. You will need to sideload so go to developer mode and turn it on.  If you want to change your username, extract the .appx file, change username.txt, then use wsappbak to repackage it. Please contribute to MinecraftConsoles for the files.
