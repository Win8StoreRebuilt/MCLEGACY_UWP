# MCLEGACY_UWP
This is MC LEGACY UWP, inspired by Minecraft UWP. This project is is available for Windows 10 1809 or higher. 

If you want to change your username, find windowsapps in program files, find the Microsoft.MinecraftLegacy_(VERSION)_x64__8wekyb3d8bbwe folder then change username.txt, then opening up the text file change to your preferred username then click file then save. Opening windowsapps save may require using powershell as admin (takeown /f "C:\Program Files\WindowsApps" /r). 

To enable sideloading, you well need to search up developer settings then enable sideloading or developer mode. 

Please contribute to MinecraftConsoles for the files.
