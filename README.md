# MCLEGACY_UWP
This is MC LEGACY UWP, inspired by Minecraft UWP. This project is is available for Windows 10 1809 or higher. 

If you want to change your username, find windowsapps in program files, Microsoft.MinecraftLegacy_(VERSION)_x64__8wekyb3d8bbwe folder then change username.txt, opening windowsapps save may require using powershell as admin (takeown /f "C:\Program Files\WindowsApps" /r). 

To enable sideloading, you well need to search up developer settings then enable sideloading or developer mode. 

Please contribute to MinecraftConsoles for the files.
