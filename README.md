# MCLEGACY_UWP
This is MC LEGACY UWP, inspired by Minecraft UWP. This project is is available for Windows 10 1809 or higher. You will need to sideload so go to developer mode and turn it on.  If you want to change your username, find windowsapps in program files, Microsoft.MinecraftLegacy_(VERSION)_x64__8wekyb3d8bbwe folder then change username.txt, opening windowsapps save may require using command prompt. Please contribute to MinecraftConsoles for the files.
